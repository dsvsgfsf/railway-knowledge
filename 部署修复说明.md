## 重新部署步骤

### 1. 更新 GitHub 仓库

在 `D:\YX\Zuoye\deploy` 目录执行：

```bash
cd D:\YX\Zuoye\deploy
git add .
git commit -m "Fix: 修复部署路径问题"
git push origin main
```

### 2. Vercel 自动部署

Vercel 会自动检测到 GitHub 的更新并重新部署。等待约 1-2 分钟即可。

### 3. 验证部署

访问你的 Vercel 域名，例如：
- 主系统: `https://your-project.vercel.app`
- 铁路局详情: `https://your-project.vercel.app/public/bureaus/index.html`
